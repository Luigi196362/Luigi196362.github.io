Política de Privacidad - Galeria Vision

Última actualización: Enero 2026

1. Introducción

Esta aplicación es un proyecto con fines exclusivamente académicos. La privacidad de los usuarios es importante, por lo que esta política explica de forma clara cómo se recopila, utiliza y almacena la información.

2. Información que se recopila

  • Imágenes capturadas con la cámara del dispositivo.
  • Imágenes seleccionadas desde la galería del dispositivo.
  • Información básica de la cuenta (usuario y contraseña).

3. Uso de la información

Las imágenes recopiladas se utilizan únicamente para:

  • Ser analizadas mediante un sistema de inteligencia artificial.
  • Clasificarlas automáticamente por categorías.
  • Mostrarlas en el perfil del usuario como una galería inteligente.

El uso de los datos es exclusivamente académico y educativo.

4. Procesamiento y almacenamiento

Las imágenes se envían a una API alojada en servicios en la nube, donde son procesadas por un sistema de inteligencia artificial.

Posteriormente, las imágenes y su categoría se almacenan en una base de datos para que el usuario pueda consultarlas desde su perfil.

5. Servicios en la nube

La aplicación utiliza servicios de backend alojados en la nube. Al tratarse de un proyecto académico, estos servicios pueden estar temporalmente inactivos o presentar interrupciones.

6. Compartición de datos

La información del usuario:

  • No se vende.
  • No se comparte con terceros.
  • No se utiliza con fines publicitarios.

7. Seguridad

Se aplican medidas básicas de seguridad acordes a un proyecto académico. No se garantiza un nivel de seguridad comercial o empresarial.

8. Cambios en esta política

Esta política puede actualizarse para reflejar cambios o mejoras en la aplicación.
